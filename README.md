# AjudanteDBA
 Ferramenta para auxiliar no trabalho do DB Admin na gestão de uma instância SQL

 ## Funcionalidades
 A primeira funcionalidade da ferramenta é apontar um banco de dados do servidor em que está conectado para ser removido. A remoçao seguirá as seguintes etapas:
	1- Fazer um backup do banco de dados;
	2- Verificar o backup e mover para uma pasta determinada;
	3- Excluir os arquivos do banco de dados;
